## Configuration

Connect Grafana to InfluxDB:
- URL: `http://influxdb:8086`
- Organization: `YOUR_ORG`
- Token: `YOUR_INFLUX_TOKEN`
- Default Bucket: `YOUR_BUCKET`

Use Flux queries to visualize processed data from UMH Core.
